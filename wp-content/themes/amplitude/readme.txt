== Description ==
Amplitude is a clean, creative and truly stunning full width WordPress theme that is specially designed to create full screen websites. This theme is a reliable platform for building a visually stimulating and delightful websites across a wide range of businesses. It equally functional as a portfolio theme that helps to improve your full-screen business website.
